# Serving Our Justice — Website

The official website for **Serving Our Justice (SOJ)**, a student-led nonprofit supporting women and survivors affected by domestic abuse through awareness, education, outreach, and advocacy.

## Why it matters

A clear, credible online presence helps survivors and supporters find resources, understand the mission, and get involved — strengthening the organization's outreach and trust.

## Tech stack

- **Framework:** Next.js (App Router) + React
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI / motion:** Framer Motion, lucide-react, clsx
- **Deployment:** Vercel

## Features

- Mission, education, and resource sections built for outreach
- Responsive, accessible layout across desktop and mobile
- Consistent visual identity (see `BRAND.md`)

## Status

Live and actively maintained.

## Run locally

```bash
npm install
npm run dev
# open http://localhost:3000
```

Production build: `npm run build` then `npm run start`.
